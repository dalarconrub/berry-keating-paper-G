# Paper G: Unconditional deterministic primality testing and other computational consequences of GRH

**Author:** David Alarcon (Universidad Pablo de Olavide, Sevilla)

## Abstract

GRH (Paper F) converts conditional complexity results into proven theorems.
Miller's deterministic primality test is now O(log^4 n) unconditionally,
faster than AKS. Generators O(log^6 q) and non-residues O(log^2 p) are
now proven bounds. No cryptographic system is affected.
